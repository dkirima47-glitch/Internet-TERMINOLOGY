Markup Language

Role: A system for annotating a document in a way that is syntactically distinguishable from the text. It is used to structure and format content on the web.

Key Point: HTML (Hypertext Markup Language) is the most common markup language used to create web pages.

Key Point: Markup languages allow developers to define elements such as headings, paragraphs, links, and images, enabling the creation of visually appealing and well-structured web content.
